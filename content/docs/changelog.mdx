Track the latest changes and updates to the API.

## December 2024

### New Endpoints

- Added 9 new API endpoints for comprehensive CPF data access
- **Calculation endpoints**: `/calculate`, `/calculate/batch`, `/projection`
- **Age group endpoints**: `/age-groups`, `/age-group/find`, `/age/from-birthdate`
- **Interest rate endpoints**: `/interest-rates`, `/interest-rates/smra`, `/interest-rates/trend`
- **Investment endpoint**: `/investment-comparison`

### Features

- URL state management with nuqs for shareable calculator links
- Interactive CPF income ceiling timeline visualisation
- Multi-year contribution projections

## September 2024

### Updates

- Updated CPF rates for 2024
- Income ceiling increased to $6,800 (effective January 2024)

## 2023

### Initial Release

- Core CPF contribution calculator
- Age-based contribution and distribution rates
- Income ceiling tracking (September 2023 increase to $6,300)
